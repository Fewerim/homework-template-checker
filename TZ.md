# Техническое задание для проекта: "Homework-Template-Checker"

## 1. Название и концепция
**HomeworkTemplateChecker** — веб-сервис для учителей, решающий проблему ручной проверки домашних заданий. Традиционно учителя тратят часы на сверку ответов учеников с эталоном. Система автоматизирует создание динамических форм ДЗ с валидацией (целые числа без десятичных, текст с ограничениями), сохраняет ответы для быстрой проверки и формирует аналитику успеваемости.

### Цель
Веб-сервис для учителей автоматизирует создание динамических форм домашних заданий с валидацией ответов. Учитель задает количество заданий и типы полей (целое число без десятичных, текст), ученики заполняют готовую форму, система сохраняет ответы для проверки учителем с автоматическим расчетом баллов.

## 2. Роли пользователей
**Гость**: просмотр демонстрационных форм заданий.

**Учитель** (расширение User): 
- Создает классы и динамические формы ДЗ (кол-во заданий + типы полей с валидацией)
- Просматривает заполненные формы учеников в таблице
- Корректирует баллы, добавляет комментарии, завершает проверку
- Видит аналитику успеваемости по классу

**Ученик** (расширение User):
- Заполняет динамическую форму ДЗ с валидацией полей (только цифры, без букв/точек)
- Получает результаты проверки с указанием ошибок и итоговой оценки
